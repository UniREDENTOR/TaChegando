# Aplicativo TaChegando

Aplicativo de mobilidade urbana, com o principal fim de permitir ao usuário acompanhar a rota do meio de transporte
que ele necessita para se locomover.


## Funcionalidades

- Login
- Criar Viagem
- Acompanhar Viagem
- Chat de Mensagens
- Acompanhamento em tempo real
- Perfil
- Buscar Viagem
